# 이 브랜치는 Dev 브랜치 입니다.







# 설정 -> 키보드 + 마우스
 - 보기 민감도 -> 마우스 조작 민감도
     - 0 ~ 1.0 까지의 소수점 으로 설정 가능
     - 기본 0.20

 - 좌우 보기 스케일
     - 0 ~ 1.0 까지의 소수점 으로 설정 가능

 - 상하 보기 스케일
     - 0 ~ 1.0 까지의 소수점 으로 설정 가능

 - 좌우 보기 반전
     - 카메라의 좌우 회전을 반전

 - 상하 보기 반전
     - 카메라의 상하 회전을 반전

 - 기본 이동
     - 기본 값 W, S, A, D -> 앞, 뒤, 왼쪽, 오른쪽 이동

 - 점프
     - 기본 값 SPACE

 - 상호작용
     - 기본 값 E

 - 장비 사용
     - 기본 값 Q

 - 질주 켜기/끄기
     - 기본 값 Ctrl

 - 주요 스킬
     - Mouse Click 1 ( 좌 클릭 )

 - 보조 스킬
     - Mouse Click 2 ( 우 클릭 )

 - 유틸리티 스킬
     - Shift

 - 특수 스킬
     - 기본 값 R

 - 정보 화면
     - 기본 값 Tab

 - 핑 보내기
     - Mouse Click 3 ( 휠 클릭 )
